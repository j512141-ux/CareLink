# 📱 手機使用指南

## CareLink 現在已完全優化為手機應用！

### 如何在手機上使用

#### **方法 1: 直接用瀏覽器打開（最簡單）**

1. 在手機上用任何瀏覽器打開應用地址：
   - **局域網**：`http://10.30.24.193:3000/`
   - **全球訪問**：使用 ngrok 提供的地址

2. 開始使用！

---

#### **方法 2: 安裝為主屏幕應用（推薦）**

這樣應用會像原生 App 一樣運行，更方便！

##### **iOS (iPhone/iPad)**
1. 用 Safari 瀏覽器打開應用地址
2. 點擊底部的 **分享** 按鈕 (⬆️)
3. 選擇 **添加到主屏幕**
4. 點擊 **添加**
5. 應用就會出現在你的主屏幕上！

##### **Android (大多數安卓手機)**
1. 用 Chrome 或其他支持的瀏覽器打開應用地址
2. 等待一會，屏幕底部會出現 **"安裝"** 提示
3. 點擊 **安裝**
4. 應用就會被安裝到你的主屏幕和應用抽屜

**如果沒有看到提示：**
1. 點擊瀏覽器菜單 (⋮)
2. 選擇 **安裝應用** 或 **添加到主屏幕**
3. 確認安裝

### ⚙️ 常見問題

**Q: 應用在哪裡可以訪問？**  
A: 只要在同一網絡或通過 ngrok 獲得的公網地址即可訪問。

**Q: 可以離線使用嗎？**  
A: 可以！基本功能會被自動緩存。但數據同步需要聯網。

**Q: 應用會自動更新嗎？**  
A: 會的。Service Worker 會自動檢查更新並在後台下載。

**Q: 怎樣卸載應用？**  
A: 
- **iOS**: 長按主屏幕圖標，選擇移除
- **Android**: 像卸載普通應用一樣操作，或長按圖標選擇移除

### 📞 需要幫助？

如果在手機上遇到任何問題，請：
1. 檢查網絡連接
2. 刷新頁面或重新啟動應用
3. 嘗試用不同的瀏覽器
4. 清除瀏覽器緩存（設置 → 隱私 → 清除數據）
