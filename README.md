# 🤝 CareLink — 社區互助應用

> 一個幫助長者和社區互相照顧、互相幫助的 PWA 應用

---

## 📖 專案簡介

**CareLink** 是一款專為長者與社區設計的互助平台。透過簡單直覺的介面，讓社區成員可以輕鬆發起邀約、提出請求，也能回應鄰里的需求，讓社區更有溫度。

---

## ✨ 功能特色

- **人臉辨識登入** — 模擬人臉掃描，自動解析使用者故事與技能
- **日記動態牆** — 分享生活點滴，按讚互動，增進社群連結
- **邀約功能（Meet）** — 發起或參與活動，例如一起吃飯、散步、泡茶
- **相助市集（Assist）** — 張貼或接受生活協助需求，如代買、搬運、陪伴
- **個人頁面** — 顯示技能徽章、幫助紀錄、親密好友列表
- **通知群組** — 支援通知全家、子女、老友等分組

---

## 🛠️ 技術架構

| 技術 | 說明 |
|------|------|
| HTML / CSS / JavaScript | 純前端，無框架依賴 |
| PWA（漸進式網頁應用） | 支援離線使用、安裝至桌面 |
| Service Worker | 快取資源，加速載入 |
| Web Manifest | 支援全螢幕模式、主畫面捷徑 |
| Google Fonts | 使用 Noto Sans TC 中文字體 |

## 🚀 如何使用

### 本地開啟

由於使用了 Service Worker，建議透過本地伺服器執行：

```bash
# 使用 Python 快速啟動伺服器
python3 -m http.server 8080

# 然後在瀏覽器開啟
http://localhost:8080
```

### 安裝為 PWA

在手機瀏覽器（Chrome / Safari）開啟後，點擊「加入主畫面」即可像 App 一樣使用。
